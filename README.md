
# TypeBloom 🌸
A cute typing speed trainer with a live keystroke heatmap.
Built with Java Swing.

## Download
Go to [Releases](link) and download:
- `TypeBloom.exe` — Windows (no Java needed)
- `TypeBloom.dmg` — Mac (no Java needed)
- `TypeBloom.jar` — Any OS (requires Java 11+)

## Run the JAR
java -jar TypeBloom.jar

## Build from source
1. Clone the repo
2. Open in Eclipse as a Java project
3. Run App.java
